# Jane Doe | Real Estate Consultant Portfolio

Professional portfolio of Jane Doe, a Visionary Real Estate Consultant and Business Development Manager.

## 🚀 Live Demo
Check out the live site here: [https://jane-deo-protofolio.vercel.app](https://jane-deo-protofolio.vercel.app)

## ✨ Features

- **🌐 Multilingual Support**: Seamlessly switch between **English** and **Arabic** translations.
- **🎨 Modern & Minimalist UI**: A premium user interface designed with a focus on typography and readability
- **📱 Fully Responsive**: Optimized for all devices, from mobile phones to high-resolution desktops.
- **🛠️ Professional Sections**:
  - **Executive Summary**: A high-level overview of professional impact and leadership.
  - **Core Competencies**: Highlighting expertise in Real Estate, Business Development, and HR.
  - **Professional Journey**: Detailed career history and key achievements.
  - **Portfolio Gallery**: Showcase of previous projects and works.
  - **Credentials**: Educational background, certifications, and a gallery of official documents.

## 📂 Project Structure

```text
├── public/                # Static assets (icons, manifest, etc.)
├── src/
│   ├── components/        # React-like components built with Vanilla JS
│   │   ├── Intro.js              # Hero section
│   │   ├── Profile.js            # Executive summary
│   │   ├── CoreExpertise.js      # List of skills
│   │   ├── Experience.js         # Career timeline
│   │   ├── PreviousWorks.js      # Portfolio items
│   │   ├── Certifications.js     # Education & Certs
│   │   └── CertificatesGallery.js # Visual certificates
│   ├── data/
│   │   └── translations.js       # Multilingual content storage
│   ├── utils/
│   │   └── canvas-wave.js        # Background animation logic
│   ├── style.css                 # Global styles and Tailwind imports
│   └── main.js                   # Application entry point and routing logic
├── index.html             # Main entry file
├── package.json           # Dependencies and scripts
└── tailwind.config.js     # Tailwind CSS configuration
```
